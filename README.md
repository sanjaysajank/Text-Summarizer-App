AI Text Summarizer App

A Flask-based web application that summarizes long articles, news reports, and research papers using Natural Language Processing (NLP).

Features

* Extractive text summarization
* Modern responsive UI
* Word & character counter
* Copy summary button
* Built with Flask, NLTK, Sumy, and Bootstrap

Technologies

* Python
* Flask
* NLTK
* Sumy
* Bootstrap
* HTML & CSS

Run the Project

pip install -r requirements.txt
python app.py

Open: http://127.0.0.1:5000
